# Отчёт о разворачивании `swagger-ui`

**Дата:** 2025-11-25

## Краткое резюме
- `swagger-ui` обычно разворачивается на основе официального образа (например `swaggerapi/swagger-ui`) и служит статические файлы интерактивной документации, подгружая `swagger.yaml`.

## Команды для воспроизведения

1) Поднять сервис:

```bash
docker compose up -d swagger-ui
docker compose logs --tail 200 swagger-ui
```

2) Проверка:

```bash
# если swagger доступен на 8080
curl -fsS http://localhost:8080
```

## Особенности
- Убедитесь, что `swagger.yaml` доступен контейнеру или что URL к спецификации указан через переменную окружения (SWAGGER_JSON / URL).
- Можно настроить Traefik для маршрутизации и TLS.
